# D2D Cover Image Requirements

## Ebook Covers
- Format: JPEG (.jpg) or PNG (.png)
- Minimum: 1600×2400 pixels
- 300 DPI recommended
- RGB color mode
- File size under 50MB

## Paperback Covers (Full Wrap)
- Format: PDF preferred, or high-res JPEG/PNG
- D2D provides a template based on trim size and page count
- Must include: front cover + spine + back cover as single image
- Bleed: 0.125" on all sides
- Spine width depends on page count and paper type
- D2D's cover calculator generates the exact template

## Paperback Trim Size Suggestions
- Middle Grade: 5×8" or 5.25×8"
- YA/Adult: 5×8" or 6×9"
- D2D Print supports: 4.25×6.875", 5×8", 5.25×8", 5.5×8.5", 6×9", 6.14×9.21", 6.69×9.61", 7×10", 7.44×9.67", 8×10", 8.5×8.5", 8.5×11"

## Series Branding

### Mycoformed Mars Saga (3 books)
- Consistent color scheme (red/orange for Mars)
- Series logo/title treatment
- Book number indicator
- Style: Clean sci-fi, bold typography

### Sporefall Cycle (3 books)
- Consistent color scheme (green/brown for fungal)
- Series logo/title treatment
- Book number indicator

### My Dog Is Judging Me (2 books)
- Consistent color scheme (fun, bright)
- Series logo/title treatment
- Book number indicator
- Style: Hand-drawn, kid-friendly, matches doodle aesthetic

### Standalone Books
- Unique cover per book
- Author name prominent
- Genre-appropriate imagery

## D2D Upload Process
1. Upload ebook manuscript (EPUB)
2. Upload ebook cover
3. Enable D2D Print for paperback
4. Use D2D's cover calculator for paperback template
5. Upload paperback cover (full wrap)
6. Set pricing for both formats
7. Publish

## Notes
- D2D provides free cover template generator for paperback
- Can also use Canva, Adobe Spark, or hire a designer
- Cover should be readable as a thumbnail (small size)
- Title and author name should be legible at small sizes
- Avoid too much fine detail (won't show at thumbnail size)
- Paperback spine text should be readable when book is on shelf
