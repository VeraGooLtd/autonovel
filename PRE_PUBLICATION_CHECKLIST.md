# PRE-PUBLICATION CHECKLIST

Manual quality checks for each book before D2D upload. Based on autonovel's evaluation framework.

## 1. SLOP DETECTION

### Tier 1: Kill on Sight
Search for these words and rewrite the sentence:
- delve → dig into, look at, examine
- utilize → use
- leverage (verb) → use, take advantage of
- facilitate → help, enable, make possible
- elucidate → explain, clarify
- embark → start, begin
- endeavor → effort, try
- encompass → include, cover
- multifaceted → complex, varied
- tapestry → describe the actual thing
- paradigm → model, approach, framework
- synergy / synergize → cooperation, combined
- holistic → complete, comprehensive
- catalyze / catalyst → trigger, spark
- myriad → many
- plethora → abundance, many

### Tier 2: Suspicious (reduce usage)
seamless, cutting-edge, innovative, streamline, empower, foster, enhance, elevate, optimize, pivotal, intricate, profound, resonate, underscore, harness, cultivate, bolster, galvanize, cornerstone, game-changer, scalable

### Tier 3: Filler Phrases
- "it's worth noting that" → just state it
- "it's important to note that" → just state it
- "importantly," → cut or restructure
- "interestingly," → cut or restructure
- "let's dive into" → start directly
- "as we can see" → just show it
- "furthermore," → use sparingly
- "moreover," → use sparingly
- "additionally," → use sparingly
- "at the end of the day" → cut
- "it goes without saying" → cut
- "when it comes to" → simplify

### Fiction AI Tells
Search for these patterns and either cut or make specific/vivid:
- "a sense of [emotion]"
- "couldn't help but feel"
- "the weight of [something]"
- "the air was thick with"
- "eyes widened"
- "a wave of [emotion] washed over"
- "a pang of [emotion]"
- "heart pounded in his/her chest"
- "raven/dark/golden hair spilled/cascaded/tumbled"
- "piercing blue/green/grey eyes"
- "a knowing smile/grin"
- "felt a surge/rush/wave of"
- "the silence was/hung/grew heavy/thick/oppressive"
- "let out a breath he/she didn't know"
- "something dark/ancient/primal stirred"

## 2. STRUCTURAL CHECKS

### Scene Outcomes (Sanderson's Promises, Progress, Payoff)
- 60%+ of middle scenes should be "Yes, but..." or "No, and..."
- "Yes, and..." should be rare (save for climax)
- Every promise in opening should pay off
- No Chekhov's Gun violations
- No deus ex machina (payoff without promise)

### Chapter-Level Checks
- Each chapter should have a clear outcome (yes/no/yes-but/no-and)
- Chapters should advance plot OR character (preferably both)
- No chapters that are purely expository
- Opening image and final image should mirror (show transformation)

### Dialogue Checks
- No two characters should sound the same
- No "as you know, Bob" exposition dialogue
- Dialogue should have subtext (characters don't say exactly what they mean)
- No speechifying (characters giving speeches to each other)
- Tags: mostly "said" and "asked", minimal adverbs

## 3. PROSE-LEVEL CHECKS

### Show Don't Tell
- "he felt angry" → show the anger through action/dialogue
- "she seemed sad" → show the sadness
- "they were nervous" → show the nervousness
- Adverbs in dialogue tags ("he said angrily") → cut the adverb, show in the dialogue itself

### Point of View
- Stay in one POV per scene (unless intentional omniscient)
- POV character can't know things they haven't observed
- Internal thoughts should be in POV character's voice

### Tense Consistency
- Pick one tense (past or present) and stick with it
- Watch for accidental tense shifts mid-paragraph

### Repetition
- No word used 3+ times in a single paragraph (unless intentional)
- No phrase repeated verbatim across chapters
- Character names: vary how you refer to characters

## 4. CHARACTER CONSISTENCY

### Naming
- Every character name should be unique within the universe
- First names should not collide with major characters from other books in the same universe
- Names should be distinct-sounding (no "John" and "Jon" in the same book)

### Character Voice
- Each major character should have a distinct speech pattern
- Vocabulary level should match character background
- No two characters should have the same verbal tics

### Character Arc
- Protagonist should change from beginning to end
- Antagonist should have clear motivation
- Supporting characters should have their own wants/needs

## 5. WORLDBUILDING CONSISTENCY

- Technology levels should be consistent
- Geography should be consistent (travel times, distances)
- Magic/science rules should not be broken without explanation
- Timeline should be trackable (no impossible date sequences)

## 6. FORMATTING

### Front Matter
- Title page with author name
- Copyright notice (no ISBN — D2D will assign)
- Table of contents (if applicable)

### Back Matter
- About the Author (50-100 words)
- Also by Daniel Ryan (list all published books)
- Social media links
- Website link

### Chapter Headings
- Consistent numbering format
- Consistent naming style
- No orphaned headings (heading with no content below)

## 7. CONTINUITY ACROSS SERIES

For multi-book sagas:
- Re-read the previous book's ending before publishing the next
- Character names, places, and events should match
- Timeline should be consistent
- No contradictions between books
